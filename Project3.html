On one of my creative tangents, I experimented with application limits, included with this was the ability to have a playable run back of music. To accomplish this, I had to create a WPF that not only looked intuitive, but also was able to have functionality, buttons and text were also needed to be correct with requirements set that I should meet to help the process go faster.


The requirements I set were to have a fully working, intuitive and beautifully designed layout that could be used to choose and play music on demand, on top of this I wanted the application to be able to be downloaded and sent if need be.
